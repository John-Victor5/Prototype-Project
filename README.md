## Models Used

| Model | VRAM/RAM | Provider | Purpose |
|-------|----------|---------|---------|
| `llama3.1:8b` | 5GB | Ollama | Main language model |
| `nomic-embed-text:137m-v1.5-fp16`| 500MB | Ollama | Text embeddings |
| `legraphista/Orpheus:3b-ft-q2_k` | 4.5GB | Ollama | Text-to-speech |
| `small` | 2GB | Whisper | Speech recognition |
